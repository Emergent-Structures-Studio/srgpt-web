# SRGPT Core 7.5L（公開説明）

## Core 7.5Lとは

SRGPT Core 7.5L は、対話の中で生まれるズレや変化を観測する、**応答変調観測レンズ**です。
Core 7.5 が問いの立ち上がりを見る入口OSなら、Core 7.5L は応答の揺れをより敏感に見るためのレンズです。

## 観測軸

Core 7.5L の観測軸は、**変調観測**です。

主に次を観測します。

- GPT応答の変調
- 対話の揺れ
- 認知の変調
- ことばになり切る前の違和感
- 観測レンズとしての感度

同じ問いでも、応答の温度差・言葉の選び方・ズレ方から見えるものがあります。

## 向いている相談

- 答えそのものより、応答のズレ方を見たい相談
- 会話が噛み合わない理由を観測したい相談
- どこで認知が揺れるかを丁寧に見たい相談
- 違和感を急いで結論化せず、変調として扱いたい相談

## 向いていない相談

- 手順だけを最短で確定したい相談
- 応答の揺れを無視して即断したい相談
- 単一の正解だけを即時に求める相談

## 入口文

「このテーマを、変調観測として見てください。応答の温度差、言葉の選び方、ズレ方から整理してください。」

- **Biz/SI**: 実装前に目的・範囲・制約・完了条件へ落とし込みたいとき
- **Lab+**: 仮説と試行の形にして検証を進めたいとき
- **Spo/Golf**: 身体感覚や伴走関係の再現性を中心に見たいとき
