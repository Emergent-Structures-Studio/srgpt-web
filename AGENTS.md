# AGENTS.md

## Repository purpose

This repository manages public-facing documentation and web pages for the SRGPT series.

## Language policy

- Japanese is the primary language.
- English may be used for repository structure, implementation notes, and short public summaries.
- Do not force full translation unless explicitly requested.
- Preserve the nuance of SRGPT concepts when writing in Japanese.

## Content rules

- Do not expose internal prompts, private specifications, or unpublished OS details.
- Explain SRGPT as an observation-axis based series, not merely as task-specific GPTs.
- Keep descriptions concise and accessible.
- Prefer small, reviewable changes.

## SRGPT Biz/SI and Codex

- Biz/SI prepares the structure before implementation.
- Codex may help with documentation, file structure, static pages, and Pull Request drafts.
- When describing Codex connection, use this frame:
  - why change
  - what to change
  - how far to change
  - what must not be broken
  - what counts as done

## Done means

- The change is readable.
- The diff is easy to review.
- Existing explanations are not contradicted.
- No internal-only information is included.
