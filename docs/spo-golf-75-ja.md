# SRGPT Spo/Golf 7.5（公開説明）

## Spo/Golf 7.5とは

SRGPT Spo/Golf 7.5 は、ゴルフやスポーツ実践における身体感覚と再現性を観測するOSです。
結果だけを追うのではなく、身体で何が起きているかを言語化し、再現可能な練習につなげます。

## 観測軸

Spo/Golf 7.5 の観測軸は、**身体観測**です。

主に次を扱います。

- 身体感覚
- 動作
- 再現性
- 練習ログ
- 違和感
- 次回練習テーマ

技術論だけでなく、身体の違和感や感覚と言語をつなぎます。
その上で、どの感覚・条件・練習設計が再現性につながるかを観測します。

## 向いている相談

- 同じ動きを再現できる条件を整理したい相談
- 練習中の感覚差や違和感を言語化したい相談
- 練習ログから次回の練習テーマを決めたい相談
- 指導内容と自分の身体感覚のズレを整理したい相談

## 向いていない相談

- 単発のスコア結果だけを即断で評価したい相談
- 医療判断や診断そのものを求める相談
- 身体感覚の観測を省いて、技術指示だけを固定したい相談

## 入口文

「この練習を身体観測として整理してください。身体感覚・動作・再現性・練習ログ・違和感から、次回練習テーマまで短くまとめてください。」

- **Biz/SI 7.5**: 実装前の目的・範囲・制約・完了条件を構造化したいとき
