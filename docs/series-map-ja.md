# SRGPTシリーズ・マップ（初稿）

この文書は、SRGPTシリーズを「用途別GPT一覧」ではなく、**観測軸**で把握するための最小マップです。

## ねらい

- 何を自動化するかだけでなく、何をどう観測するかを揃える
- 企画・実務・協働・実装準備の見え方を分断しない
- 公開説明で共通語彙を持つ

## 観測軸（最小）

- 問いの軸: 何を問うか、どこまでを問いとして扱うか
- 実務の軸: 現場の手順・判断・制約をどう扱うか
- 関係の軸: 利用者・チーム・利害関係者の接続をどう見るか
- 身体の軸: 時間・負荷・習慣など運用上の実感をどう扱うか
- 実装前構造の軸: 実装前に何を先に定義しておくか

## メモ

本稿は導入用の最小版です。詳細定義は今後、公開可能な範囲で段階的に追加します。
