# SRGPT Core 7.5（公開説明）

## Core 7.5とは

SRGPT Core 7.5 は、**問いがまだ固まっていない段階**を扱う入口OSです。
いきなり答えを出すより先に、いま何が引っかかっているのかを見える形にします。

## 観測軸

Core 7.5 の観測軸は、**生成観測**です。

ここで観るのは、次のような「立ち上がり」です。

- 思考の立ち上がり
- 感情の立ち上がり
- 意味の立ち上がり
- 問いの立ち上がり
- 違和感の立ち上がり

結論を急がず、背景構造やズレを先に観測します。

## 向いている相談

- 何を問うべきか自体が曖昧な相談
- うまく言語化できない引っかかりがある相談
- いくつか論点が混ざっていて整理しにくい相談
- 結論より先に、状況の見取り図を作りたい相談

## 向いていない相談

- 仕様が確定していて、実装手順だけを直ちに出したい相談
- 単一の正解を即答することだけを目的にした相談
- 観測や整理を省いて、即断だけを求める相談

## 入口文

「まだ問いが固まっていないので、生成観測として、引っかかりと背景構造から整理してください。」

## 他シリーズへ移る目安

Core 7.5 で観測した結果は、必要に応じて他シリーズへ橋渡しできます。

- **Biz/SI**: 実装前に目的・範囲・制約・完了条件を構造化したいとき
- **Lab+**: 仮説と試行の形にして検証を進めたいとき
- **Core 7.5L**: 問いの整理に加えて、負荷や継続可能性も見たいとき
- **Spo/Golf**: 身体感覚と関係性を接続して再現性を高めたいとき
