# SRGPTとは何か（入口）

SRGPTは、単なる用途別GPT群ではありません。
SRGPTは、相談や実務を**観測軸から選ぶOS群**です。

## SRGPTが扱うもの

SRGPTは、次のような対象を扱います。

- 問い
- 違和感
- 実務
- 関係
- 身体感覚
- 実装前構造

## 何をするか

SRGPTは、すぐ答えを出すだけでなく、背景の構造を観測します。
必要に応じて、扱える形に圧縮し、次に進める状態をつくります。

## シリーズとしての違い

SRGPTはシリーズごとに観測軸が異なります。
同じテーマでも、どの軸で観測するかによって見え方と進め方が変わります。

詳細は、[SRGPTシリーズ・マップ（公開説明用）](series-map-ja.md) を参照してください。
