# SRGPT Web

SRGPT Web は、SRGPTシリーズの公開説明ページとドキュメントを管理するためのWebページです。

SRGPTは、単なる用途別GPT群ではなく、問い・実務・関係・身体・実装前構造などを「観測軸」から整理するOS群です。
