# SRGPT Biz/SI 7.5（公開説明）

## Biz/SI 7.5とは

SRGPT Biz/SI 7.5 は、実務やSI案件で起きる要素を、実装前に整理するためのOSです。
答えを即断するより先に、案件の構造を観測し、判断前の土台をそろえます。

## 観測軸

Biz/SI 7.5 の観測軸は、**構造観測**です。

主に次を扱います。

- 案件
- 業務
- 制度
- 責任
- リスク
- 判断順序

SI文脈では、次を重点的に整理します。

- 要件
- 責任分界
- 依存関係
- 工程
- 受入条件

## 向いている相談

- 案件開始前に、目的・範囲・制約・完了条件をそろえたい相談
- 要件や責任分界が曖昧で、後工程の手戻りリスクが高い相談
- 関係者ごとの判断順序をそろえて、説明可能性を上げたい相談
- 実装に入る前に、SI案件の構造を短く可視化したい相談

## 向いていない相談

- 経営判断そのものの代行を求める相談
- 契約判断そのものの代行を求める相談
- 管理責任の引き受けを求める相談
- 構造整理を省いて、即時の実装手順だけを確定したい相談

## Codex接続との関係

Biz/SI 7.5 は、Codexで実装に入る前の準備層です。
接続時は、次の5点を先に整えます。

- なぜ変えるのか
- 何を変えるのか
- どこまで変えるのか
- 何を壊してはいけないのか
- 何をもって完了か

この5点をそろえることで、変更の境界とレビュー観点を保ったまま実装へ渡しやすくなります。

## 入口文

「この案件を構造観測として整理してください。目的・範囲・制約・完了条件と、要件・責任分界・依存関係・工程・受入条件を先にそろえてください。」

## 他シリーズへ移る目安

- **Core 7.5**: そもそもの問いや論点が曖昧で、先に問いを整えたいとき
- **Core 7.5L**: 対話のズレや認知の揺れを追加で観測したいとき
- **Lab+ 7.5**: 試行と学習ループで仮説検証を回したいとき

Biz/SI 7.5 は、判断を代行するOSではありません。
実務判断の前に、構造を見える形にそろえるためのOSです。
